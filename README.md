# flutter-getx-modern-architecture

A Flutter GetX Architecture for starter who need a architecture, which applied in real project.


## Project structure
```
|-- lib
    |-- app.dart
    |-- main.dart
    |-- root.dart
    |-- config
        |-- app_constants.dart
        |-- app_colors.dart
        |-- app_text_styles.dart
    |-- controllers
    |-- extensions
    |-- pages
    |-- providers
        |-- database
        |-- network
            |-- apis
            |-- models
            |-- api_endpoints.dart
            |-- api_provider.dart
            |-- api_representable.dart
    |-- repositories
    |-- utilities
    |-- views
```
